## How to Use the Right-Click Menu

Right-click anywhere in this editor or the preview pane to access the custom context menu!

### Main Menu (Markdown Actions)

When you right-click, you'll see these markdown editing options:

1. **Cut** (Ctrl+X) - Cut selected text
2. **Copy** (Ctrl+C) - Copy selected text  
3. **Paste** (Ctrl+V) - Paste from clipboard
4. **Select All** (Ctrl+A) - Select all text
5. **Bold** (Ctrl+B) - Make text bold
6. **Italic** (Ctrl+I) - Make text italic
7. **Inline Code** (Ctrl+`) - Format as code
8. **Insert Link** (Ctrl+K) - Add a hyperlink
9. **Insert Image** (Ctrl+Shift+I) - Add an image
10. **Show more options** - Access browser-like features ‚ñ∂

## Why Two Menus?

### Main Menu (Immediate Access)
Shows **markdown editing actions** you'll use most often:
- Cut, Copy, Paste, Select All
- Bold, Italic, Code
- Insert Link, Insert Image

### Extended Menu (Browser Actions)
Shows **browser-like features** for full functionality:
- Navigation: Back, Forward, Reload
- File operations: Save, Print
- Developer tools: View Source, Inspect
- Common actions: Find, Undo, Redo

## Tips

- The menu auto-positions to avoid going off-screen
- Click anywhere outside the menu to close it
- The menu closes automatically after selecting an action
- Use "Show more options" for browser-like commands
- Try keyboard shortcuts for faster workflow

## Security Note

The default browser right-click is disabled to prevent context menu security issues. This custom menu provides all the functionality you need while maintaining security. If you need native browser features, use keyboard shortcuts (F12 for DevTools, Ctrl+P for Print, etc.).
