Lancer Notes Documentation

Welcome to Lancer Notes - a powerful, modern Markdown editor with real-time preview and comprehensive settings management. Features include advanced table operations, auto-save functionality, zoom controls, and a complete settings system.

Getting Started

Lancer Notes provides an intuitive interface for creating and editing Markdown documents. The editor is split into two panes: the editor on the left and the live preview on the right.

üí° Pro Tip: Both the editor and preview panes are editable! You can type in either pane and changes sync automatically.

Markdown Basics

Markdown is a lightweight markup language that lets you format text using simple syntax. Here are the essentials:

Text Formatting

**Bold Text** - Wrap text with double asterisks
*Italic Text* - Wrap text with single asterisks
~~Strikethrough~~ - Wrap text with double tildes
`Code` - Wrap text with backticks
~Subscript~ - Wrap text with single tildes
^Superscript^ - Wrap text with carets

Lancer Notes Structured Text (.st)

Lancer Notes Structured Text (LNST) is the enhanced Markdown format used by Lancer Notes. It keeps standard Markdown readable while unlocking richer features when opened in the editor.

Core Principles

  • Markdown foundation for headings, emphasis, lists, links, and images
  • Progressive enhancement: features activate when extensions are enabled
  • Readable source text that degrades gracefully outside Lancer Notes

File Details

  • Extension: .st
  • MIME type: text/x-lancer-structured-text
  • UTF-8 encoding recommended
  • Supports auto-save, Save, Save As, and Import flows in v2.2+

Enhanced Features

  • Tables with optional formatting flags (striped, no-borders, header-highlight)
  • Image positioning via {left}, {right}, {center} and info badges
  • Footnotes, definition lists, superscript/subscript, and highlights (Extended Markdown)
  • Task list checkboxes (Extended Markdown or GFM)
  • Syntax-highlighted code blocks when the Prism extension is enabled

Saving & Importing: Use File ‚Üí Save or Save As to write notes with a .st extension. The editor can open .st, .md, and .txt files via File ‚Üí Open or drag-and-drop.

LNST documents remain compatible with standard Markdown readers‚Äîextended elements appear as readable fallbacks. Inside Lancer Notes, enable extensions under Extensions ‚Üí Manage to unlock superscripts, footnotes, and other advanced behaviors.

Links & Images

[Link text](https://example.com)
![Alt text](image-url.jpg)

# Image Positioning
![Alt text](image.jpg){left}    # Float image to the left
![Alt text](image.jpg){right}   # Float image to the right
![Alt text](image.jpg){center}  # Center the image

# Clear floats after wrapped images
---clear---

Image Auto-Resize: Enable auto-resize in the image dialog to automatically optimize image sizes for your document.

Image Positioning: Add {left}, {right}, or {center} after the image syntax to control positioning. Use ---clear--- to clear floats and ensure content appears below wrapped images.

‚ö†Ô∏è Compatibility Notice: Image positioning syntax ({left}, {right}, {center}) is a custom extension designed for Lancer Notes and is NOT part of standard Markdown or GitHub Flavored Markdown. This feature is specifically designed to make document layout easier for students and educational use. If you export your markdown to other editors or platforms, these positioning tags will not work and should be removed for compatibility.

Advanced Features

Auto-Save Functionality

Automatically save your work to localStorage every 30 seconds. Never lose your content again with configurable auto-save intervals.

Zoom Controls

Adjust text size with zoom in/out buttons and keyboard shortcuts (Ctrl +/-/0). Zoom level persists between sessions for consistent readability.

Word Wrap

Toggle word wrap to prevent horizontal scrolling. Works in both editor and preview panes. Code blocks remain scrollable for better readability of long code lines.

Image Auto-Resize

Automatically resize images to fit your document. Configure maximum width via Image Cap button.

Find & Replace

Powerful search with regex support, case sensitivity options, and match highlighting.

Undo/Redo History

Hover over undo/redo buttons to see your edit history and jump to any previous state.

Live Preview Editing

Edit directly in the preview pane. Changes sync automatically with the markdown editor.

Right-Click Context Menu

Custom context menu with markdown actions and browser-like features. Mimics standard browser right-click (Back, Forward, Reload, Print, Inspect) for security while providing quick markdown formatting.

Tips & Tricks

Optimize Your Workflow

  • Hover over undo/redo buttons to access your edit history
  • Enable word wrap for better readability of long lines
  • Use the Find & Replace dialog for bulk text operations
  • Toggle between split, editor-only, and preview-only modes with Ctrl+1/2/3
  • Enable image auto-resize to keep your document responsive
  • Configure auto-save in settings to never lose your work
  • Use the comprehensive table dropdown for advanced table operations
  • Export your settings to backup preferences across devices
  • Use Save As for custom file names and Print for formatted output
  • Use zoom controls (Ctrl +/-/0) to adjust text size for better readability
  • Access the modern settings interface with Ctrl+, for comprehensive configuration
  • Choose between Light, Dark, or Auto themes in the General settings section
  • Clear search history from the Data & Privacy section in settings
